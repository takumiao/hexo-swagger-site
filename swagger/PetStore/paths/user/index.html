[Post] user - Create user

This can only be done by the logged in user.


RequestBody - Created user object

application/json   ( object<User> )

  • id   ( integer {$int64} )
  • username   ( string )
  • firstName   ( string )
  • lastName   ( string )
  • email   ( string )
  • password   ( string )
  • phone   ( string )
  • userStatus   ( integer {$int32} ) - User Status

Response

default - successful operation
