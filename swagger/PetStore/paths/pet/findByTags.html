[Get] pet/findByTags - Finds Pets by tags

Muliple tags can be provided with comma separated strings. Use tag1, tag2, tag3 for testing.


Parameters

tags   ( array ) - Tags to filter by

  • items   ( string )

Response

200 - successful operation

application/xml   ( array )

  • items   ( object<Pet> )
    • id   ( integer {$int64} )
    • category   ( object<Category> )
      • id   ( integer {$int64} )
      • name   ( string )
    • name   ( string )
    • photoUrls   ( array )
      • items   ( string )
    • tags   ( array )
      • items   ( object<Tag> )
        • id   ( integer {$int64} )
        • name   ( string )
    • status   ( string ) - pet status in the store

application/json   ( array )

  • items   ( object<Pet> )
    • id   ( integer {$int64} )
    • category   ( object<Category> )
      • id   ( integer {$int64} )
      • name   ( string )
    • name   ( string )
    • photoUrls   ( array )
      • items   ( string )
    • tags   ( array )
      • items   ( object<Tag> )
        • id   ( integer {$int64} )
        • name   ( string )
    • status   ( string ) - pet status in the store

400 - Invalid tag value
